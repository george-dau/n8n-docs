April 20, 2026

#### Dashboards:

We continued adding functional improvements and visual enhancements to expand how customers can visualize Worlds' data:

- **Tracks Performance Optimization** — Optimized track summary queries to reduce latency and improve responsiveness across the dashboard experience.
- **Dashboard Loading Polish** — Refined the CSS-based loading system to eliminate jerky transitions, ensuring dashboard loads feel smooth and intentional with no abrupt widget pop-in.
- **Enable Agent Name Filtering** — Users can now filter Events Activity by Agent name instead of only by ID, improving usability.
- **Bar Chart Enhancements** — Enhanced bar charts with a glow effect for dominant values and improved background treatments for better readability.

#### Visual Search:

We expanded visual similarity search to display all detected objects, so that investigations never hit a dead end:

- **Show All Detections for "More Like This" Searches** — When using selection mode, the system now displays all detections visible in the video segment (not just the selected track) allowing users to click on any visible detection to launch a new search.

#### Agentic AI:

We added more UI functionality to support and provide additional insight into the data that AI can analyze and return:

- **Tracks Area Chart** — Introduced a new line area chart visualizing detection trends over time with gradient styling and key summary stats including total detections, averages, peaks, and valleys.
- **Event Search Graph** — Resolved issues with event search graphs including filter accuracy, timezone-correct time ranges, and improved hover interactions.
