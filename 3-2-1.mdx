title: "3.2.1"
---

May 1, 2026

---

#### Agentic AI

We continued expanding Agentic AI with smarter responses and richer interactivity:

- **Conversational Sessions** — Users can now engage with Agentic AI in multi-turn sessions, to refine queries and dig into insights.
- **Upgraded AI Engine** — The underlying AI analysis engine has been updated, delivering more accurate, higher-quality responses across all agentic workflows.

---

#### Dashboards

We continued adding functional improvements and visual enhancements to expand how customers can visualize their data:

- **Custom Bucketing** — Custom bucketing for all data types enables shift-comparison by custom time-ranges, detection groups to visualize categories in consolidated view, and more.
- **Categorical Timeseries** — Dashboards now support categorical timeseries data, enabling richer activity comparison across detection types and event categories.
- **Expanded Widget Filtering** — Filtering is now supported on additional widget types, giving users more control over the data surfaced in their dashboards.
- **Total Count Widget Improvements** — The Total Count widget has been upgraded with improved formatting and statistical calculations (average, mean, and median).

---

#### Builder Enhancements

We made several usability improvements to Site Builder to support faster, more intuitive site configuration:

- **Redraw Zones** — Users can now redraw camera-based zones from scratch without disrupting existing workflow references.
- **Navigation Improvement** — Double-clicking any list item in Site Builder now zooms the map to that element, making it faster to navigate large or complex sites.

---

#### Bug Fixes & Performance Improvements

The following issues have been resolved in this release:

- **Tracking Performance** — Implemented several performance improvements improving track consistency for static and slow-moving objects.
- **Tag Color Editing** — Resolved an issue that prevented users from editing a tag's color after it had been created.
- **Event Detail Video Playback** — Fixed video playback errors occurring in the Event Detail view.
